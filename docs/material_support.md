# Material Support

V1 targets glTF-friendly Principled BSDF workflows.

## Supported

- Base Color
- Metallic
- Roughness
- Alpha
- Emission color
- Emission strength
- Clearcoat / Coat Weight
- Clearcoat Roughness / Coat Roughness
- Transmission / Transmission Weight
- IOR

## Supported texture-linked workflows

- Base Color image texture
- Roughness image texture
- Metallic image texture
- Normal map texture
- Alpha texture
- Emission texture

## Unsupported in V1

- Arbitrary Cycles node graphs
- Procedural texture graph translation
- Custom shader recreation in Three.js
